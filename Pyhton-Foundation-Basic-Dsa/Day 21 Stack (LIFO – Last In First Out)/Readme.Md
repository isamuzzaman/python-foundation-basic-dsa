# Day 21: Stack
# Description

This day focuses on understanding the Stack data structure and implementing it using Python. Learned the LIFO (Last In First Out) principle and applied stack to solve a real problem (Balanced Parentheses).

# Summary

Learned what a Stack is and how it works

Implemented stack using Python list

Built a custom Stack class with push, pop, peek, and is_empty methods

Solved the Balanced Parentheses Checker problem using stack

# Skills Gained

Understanding LIFO principle

Implementing data structures from scratch

Using stack in real-world logic problems

Thinking about time and space complexity
