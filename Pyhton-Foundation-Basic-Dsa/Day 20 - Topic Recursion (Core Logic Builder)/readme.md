# Day 20: Recursion 
# Description

This day focuses on understanding recursion in Python, including base cases, call stack, and preventing infinite recursion. Applied recursion to solve common programming problems.

# Summary

Learned what recursion is and how it works

Understood base cases and call stack

Solved problems like factorial, Fibonacci, sum of digits, reverse string, and palindrome check

Practiced mental tracing of recursive calls

# Skills Gained

Writing recursive functions

Understanding and visualizing function call stack

Solving problems recursively instead of iteratively

Handling edge cases and avoiding infinite recursion
